iPhone و iPad — Safari

مهم: من Safari فقط (ليس Chrome داخل آيفون).

  1. افتح الرابط الرسمي للموقع في Safari{% if install_url %} ({{ install_url }}){% endif %}.
  2. زر مشاركة (مربع وسهم للأعلى).
  3. إضافة إلى الشاشة الرئيسية.
  4. إضافة في الزاوية.

بعدها تظهر أيقونة «{{ theme.site_name }}» على الشاشة الرئيسية.

Android — Chrome أو Edge

  1. افتح الموقع في Chrome أو Edge.
  2. من الشريط السفلي: تثبيت إن ظهر، أو
  3. القائمة ⋮ ← تثبيت التطبيق / Install app.

كمبيوتر — Windows / Mac

  1. Chrome / Edge: أيقونة التثبيت ⊕ في شريط العنوان، أو القائمة ← تثبيت التطبيق.
  2. Safari (Mac): ملف ← إضافة إلى Dock.

لم يظهر زر التثبيت؟

  • تأكد أن الموقع يفتح بـ https:// (ليس http فقط).
  • على السيرفر: شغّل npm run assets ثم ارفع مجلد static/icons/ (ملفات PNG).
  • أعد تحميل الصفحة بدون كاش: Ctrl+F5 أو مسح بيانات الموقع.
  • على iPhone: لا يظهر زر تلقائي — اتبع خطوات Safari أعلاه (هذا طبيعي).
